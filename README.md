# projeto_mystock
Sistema de controle de estoque simples desenvolvido em HTML, CSS e JavaScript, com funcionalidades de cadastro e remoção de produtos.

---

# Sistema de Controle de Estoque — Maricota Wear

Este é um sistema simples de controle de estoque EM DESENVOLVIMENTO com **HTML, CSS e JavaScript**.  
O projeto simula um painel administrativo para gestão de produtos de uma loja de vestuário.

---

## 🚀 Funcionalidades

- Cadastro de produtos (nome e quantidade)
- Listagem dinâmica dos itens em estoque
- Remoção de produtos
- Atualização automática da interface

---

## 🖥️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (EM FASE DE DESENVOLVIMENTO)
